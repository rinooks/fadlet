# 00. 제품 개요

> Fadlet의 비전·타겟·차별화·기술 스택 등 **모든 Phase에 공통되는 컨텍스트**.

---

## 1. 제품 정체성

### 정의
**Fadlet = Facilitator-friendly Padlet**

워크숍 운영자(Facilitator)를 위한 협업 보드 SaaS. 한국 기업교육 워크숍 환경에 최적화.

### 비전
> "운영자가 더 잘 진행하면, 학습자는 더 깊이 참여한다."

### 미션
워크숍 진행자도, 참여자도 5분 안에 익숙해지는 도구를 만든다. 학습은 도구가 아니라 콘텐츠와 사람에서 일어나야 한다.

### 핵심 가치 제안
1. **즉시 참여** — 회원가입 없이 6자리 코드로 30초 안에 합류
2. **HRD 특화** — KPT, 4F, 9 Window 등 한국 워크숍 표준 템플릿 내장
3. **실시간 소통** — 보드 옆 채팅 패널로 즉각적인 질문·반응 교환
4. **한국 환경 최적화** — 한국어 UI, 토스페이먼츠, 세금계산서

---

## 2. 타겟 페르소나

### Primary: 김교육 매니저 (35세, 워크숍 운영자)
- 대기업 HRD 매니저, 월 2-3회 사내 워크숍 직접 진행
- 패들렛 사용 중이지만 한국어 어색함과 결제 불편으로 대안 탐색
- **주요 페인**: 워크숍 시작 전 가입 안내에 5-10분 소비, 운영 중 단계 전환과 즉석 질문 응답을 동시에 해야 함

### Secondary: 이학습 책임 (28세, 워크숍 참여자)
- 일반 직원, 회사 워크숍 참여
- IT 친숙도 보통, 회원가입과 로그인을 가장 싫어함
- 모바일 접속 비중 높음 (60% 이상)
- **주요 페인**: 워크숍 참여를 위해 또 다른 계정 생성 부담, 질문이 있어도 손 들기 부담스러움

### Decision Maker: 박관리 팀장 (45세, 인재개발팀장)
- 도구 구매 결재 권한
- 보안·세금계산서·기업 단위 라이선스 관리에 민감
- **주요 페인**: 해외 SaaS 결제 시 회계 부서와 마찰, 데이터 보안 우려

## 3. 핵심 차별화 (vs Padlet)

| 영역 | Padlet | Fadlet |
|---|---|---|
| 참여 진입 | 회원가입 또는 인증 | 코드 + 닉네임만 |
| 한국어 UI | 기계 번역체 | HRD 현업 용어 |
| 실시간 채팅 | 댓글만 가능 | 전용 채팅 패널 |
| 워크숍 템플릿 | 일반 보드만 | KPT · 4F · 9 Window 내장 |
| 운영자 모드 | 없음 | 타이머 · 단계 · 모더레이션 |
| 결제 · 세금계산서 | 해외 결제 | 토스페이먼츠 + 세금계산서 |

### 포지셔닝 한 문장
> "패들렛은 참여자 중심, Fadlet은 운영자 중심"

## 4. 기술 스택

### 프론트엔드
- **Next.js 14** (App Router) + **TypeScript** (strict mode)
- **Tailwind CSS** + **shadcn/ui**
- **Zustand** (클라이언트 상태)
- **TanStack Query** (서버 상태)
- **zod** (스키마 검증)

### 백엔드 (Firebase)
- **Firestore** — 메인 DB, 실시간 동기화 (`onSnapshot`)
- **Firebase Authentication** — Anonymous Auth 중심, 운영자만 이메일/Google
- **Firebase Storage** — 이미지·파일 업로드 (Phase 2부터)
- **Firebase Functions** — 서버사이드 로직 (Phase 3, 결제·OG 메타데이터 등)

### 인프라
- **Vercel** — 호스팅 + Edge Functions
- **GitHub** — 저장소 + Vercel 자동 배포 연동
- **Vercel Analytics** + **Posthog** — 사용자 행동 분석

### Phase 3 추가
- **토스페이먼츠** — 국내 결제·정기결제·세금계산서

## 7. GTM 전략

핵심 메시지: **"운영자를 위한 보드, Fadlet"**

1. **검증** — 레퍼런스HRD 자체 워크숍에서 사용
2. **베타** — 기존 고객사(현대·기아·LG·삼성 등) HRD 부서에 무료 제공
